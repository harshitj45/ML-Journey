# Student Grade Management System

## Description
A menu-driven CLI application to manage
student grades, generate report cards,
rankings, and class statistics.
Built using only core Python — no libraries.

## Topics Used
- Variables & Data Types
- Strings & f-strings
- Lists, Tuples
- Dictionaries & Sets
- List/Dict Comprehensions
- For & While Loops
- Functions

## Features
- View all students summary table
- Individual detailed report cards
- Class rankings with medals
- Class statistics & grade distribution
- Subject-wise performance analysis
- Class topper finder

## How to Run
````bash
python grade_system.py
````
